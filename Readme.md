# Методичка `Markdown`
### Файли
Для створення файлів `Markdown` на Github файли повинні завершуватися розширенням `.md`
### Форматування
| Написання | Вигляд | Що означає |
|---|---|---|
| `*sometext*` | *sometext* | Текст робиться курсивом |
| `**sometext**` | **sometext** | Текст робиться жирним |
| `~~sometext~~` | ~~sometext~~ | Текст робиться перекреслений |
| `<sub>sometext</sub>` | <sub>sometext</sub> | Текст робиться індексним |
| `<sup>sometext</sup>` | <sup>sometext</sup> | Текст робиться степенем |
| `<ins>sometext</ins>` | <ins>sometext</ins> | Текст робиться підкресленим |

Для створення цитат використовують `>`

```
Це не цитата  
> Це цитата
```
Це не цитата  
> Це цитата

Для вставлення **коду** використовується `` ` ``  
Якщо потрібно виділити одне слово/фразу ми її огортаємо **один** раз  
Але якщо потрібно огорнути фрагмент тексту - огортаємо **три*** рази  

Наприклад
````
Тут буде виділений `це` слово
```
А тут весь фрагмент
```
````
Тут буде виділений `це` слово
```

```
